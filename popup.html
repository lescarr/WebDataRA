Twitter Research Assistant

Data will be collected from Twitter that you can save into Excel for further analysis.

  1. On a Twitter search results page or on a Facebook timeline page, the browser will continually scroll to the bottom of the page to trigger more results, until the page stays static for more than 5 seconds.
  2. Data will also be extracted from a search results on Google, Google Scholar, Github, SSRN, and Core. Press the button on each page of the search results and the data will be collected into a single table for convenience.

For more information about using this software, see the author's WebDataRA page. Any questions or comments, please refer to Leslie Carr. Please be sure to treat the data according to the terms and conditions of the source.
